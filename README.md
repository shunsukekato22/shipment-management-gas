# 出荷管理システム（GAS）

## 概要

Googleスプレッドシート上でチェックボックスをONにすると、
出荷日ごとのシートに自動転記されるシステムです。

---

## ファイル構成

```
入力用スプレッドシート（SRC）
　└ 入力用シート　　　　　← チェックボックスで操作

出荷管理スプレッドシート（DST）
　├ 原紙シート　　　　　　← 新規シート作成時のテンプレート
　├ 権限管理シート　　　　← 管理者・担当者のメールアドレス管理
　└ yy/MM/dd シート　　　 ← 出荷日ごとに自動生成
```

---

## 機能一覧

| 機能 | 説明 |
|------|------|
| 自動転記 | チェックONで出荷日シートに転記 |
| 自動削除 | チェックOFFで転記データを削除 |
| シート自動生成 | 出荷日シートが無ければ原紙からコピーして作成 |
| シート並べ替え | 日付順に自動ソート |
| 自動ロック | 当日以前のシートを自動で保護 |
| 土日スキップ | 金曜日は翌営業日（月曜）基準でロック判定 |

---

## セットアップ手順

### 1. スクリプトをコピーする

GASエディタに以下の関数をコピーする。

- `onEditHandler`
- `sortSheetsByDate`
- `protectPastSheets`
- `setConfig`（初回のみ使用・実行後削除）

### 2. スプレッドシートIDを登録する

`setConfig` 関数内の指定箇所に出荷管理スプレッドシートのIDを入力し、
GASエディタで `setConfig` を選択して実行する。
完了後、`setConfig` 関数は削除する。

#### スプレッドシートIDの確認方法

```
https://docs.google.com/spreadsheets/d/【ここがID】/edit
```

### 3. トリガーを設定する

| 関数 | トリガー種別 | 設定値 |
|------|-------------|--------|
| `onEditHandler` | スプレッドシートから → 編集時 | － |
| `protectPastSheets` | 時間主導型 | 毎日・任意の時刻 |

---

## シート設定

### 入力用シート（SRC）

| 列名 | 説明 |
|------|------|
| 転記済み | チェックボックス。ONで転記、OFFで削除 |
| 出荷日 | 出荷日（Date型） |
| その他 | 転記対象の項目 |

### 権限管理シート（DST）

| 列 | 内容 |
|----|------|
| A列 | メールアドレス |
| B列 | 役割（`管理者` or `担当者`） |

- **管理者**：ロック後もシート全体を編集可能
- **担当者**：ロック後も一部セル（K・L・P〜S列）のみ編集可能

---

## ロック仕様

### ロック対象

当日を基準に翌営業日より前のシートを自動ロック。

| 実行日 | ロック対象 |
|--------|-----------|
| 月〜木曜 | 翌日以前のシート |
| 金曜 | 翌月曜以前のシート |

### 編集可能範囲（ロック後）

| 権限 | 編集可能範囲 |
|------|-------------|
| 管理者 | シート全体 |
| 担当者 | K列・L列・P〜S列 |

---

## 注意事項

- 土日の出荷日は設定しない運用を前提としています
- `setConfig` 実行後は必ず関数を削除してください
- スクリプトは入力用スプレッドシート（SRC）のGASに設置してください
- スプレッドシートIDは機密情報です。READMEや共有ドキュメントに直接記載しないでください
